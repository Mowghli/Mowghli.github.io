Portfolio

VASSL: A Visual Analytics Toolkit for Social Spambot Labeling

Developed a web toolkit with JavaScript with D3.js framework, that uses Twitter data about real and fake users and generates visualizations that help analysts understand the relative number of Twitter spambots over the years. Made the Time View: a box and whisker plot that shows the growth of spambots in Twitter through the years.

Wildlife Detection Application

Collaborated on the development of an Android app with Kotlin framework, that takes photos and identifies animal and plant species in photos taken. This is useful in remote areas to identify unknown animals and plants. It integrates Firebase Storage and Firestore Database to store images and metadata. The Machine Learning model was trained and deployed on Google AutoML Vision, which services requests from the app.

Video Surveillance Application on cloud (AWS)

Deployed a cloud app with Java and Spring boot framework on Elastic Compute Cloud (EC2), that applies deep learning to detect objects in videos collected from Internet of Things (IoT) devices in real-time. AWS services like Simple Queue Service (SQS), and Simple Storage Service (S3) were also used. Created the app-tier instances that process videos sent from clients. The application handles concurrent client requests, performs load balancing, and scales up or down, depending on the number of requests.

Determining User Demographics based on Browsing History

Worked on a desktop app with Java and Oracle 11g, that takes in user’s locally stored browsing history as input and predicts the demographic attributes of the user like gender, age group, and areas of interest. Integrated a tag cloud that represents the relevant areas of interest. The application aims to help advertisement companies analyze user demographics and provide relevant advertisements to users, to sell more products and services.
